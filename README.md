### Featured Projects

**[AudioGuard](https://github.com/sincera315/AudioGuard-)**
Forensic deepfake-audio detection system using AASIST3 and XLS-R 300M (Wav2Vec2), with Grad-CAM explainability to visualize manipulated acoustic regions for forensic reporting.

**[Story2Audio Microservice](https://github.com/sincera315/Pretrained_Story_to_Audio_Model_Integration_CPU-GPU)**
Production-ready gRPC microservice for high-fidelity text-to-audio generation using Suno TTS — containerized with Docker and deployed via a GitHub Actions CI/CD pipeline, cutting release time from ~20 min to under 4 min.

**[Agentic Flight Booking System](https://github.com/sincera315/Assignment3_Agentic_AI_N8N)**
Autonomous agent built with n8n and Docker that automates end-to-end flight search and booking through API integrations and multi-step reasoning loops.
